# Music-Emotion-Dataset
A music dataset comprising six emotional categories, containing a total of 3,042 samples each lasting 20 seconds. It includes 506 angry tracks, 508 fearful tracks, 507 happy tracks, 506 sad tracks, 508 sleepy tracks, and 507 uplifting tracks.
Dataset DOI :10.57760/sciencedb.29610
The dataset is still under review, will soon be available.
